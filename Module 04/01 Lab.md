## Теоретическая справка

### 1. Справочные системы Linux

| Ресурс | Описание | Пример использования |
|--------|----------|---------------------|
| `--help` | Краткая справка по команде (встроена в большинство утилит) | `ls --help` |
| `man` | Система руководств (manual pages) — подробное описание команд, файлов, системных вызовов | `man ls` |
| `info` | Более детальная документация в гипертекстовом формате (часто содержит примеры) | `info ls` |

### 2. Структура man-страниц

Man-страницы разделены на 8 секций:
1. Пользовательские команды
2. Системные вызовы
3. Библиотечные функции
4. Специальные файлы (устройства)
5. Форматы файлов
6. Игры
7. Разное
8. Команды администрирования

### 3. Навигация в man и info

| Действие | man | info |
|----------|-----|------|
| Поиск вперёд | `/` + слово | `/` + слово |
| Поиск назад | `?` + слово | `?` + слово |
| Следующий раздел | `n` (для поиска) | `n` |
| Выход | `q` | `q` |

### 4. Полезные команды для поиска

```bash
man -k <ключевое_слово>   # Поиск по описанию (whatis)
apropos <ключевое_слово>  # Аналог man -k
whatis <команда>          # Краткое описание команды
```

## Решение (эталон выполнения)

```bash
# ============ ЗАДАНИЕ 1 ============
ls --help                    # вывод краткой справки
# Опция -a или -A для показа скрытых файлов

mkdir --help                 # справка для mkdir
# Опция -p для создания родительских каталогов

# ============ ЗАДАНИЕ 2 ============
man grep                     # открыть man-страницу
# Поиск: /-i
# Поиск: /ignore case

man passwd                   # откроется секция 1 (пользовательская команда)
man 5 passwd                 # откроется секция 5 (формат файла)

whatis tar                   # краткое описание tar

# ============ ЗАДАНИЕ 3 ============
info find                    # открыть info-документацию
# Навигация: стрелки, Enter для перехода по ссылкам
# Поиск: /-name
# Выход: q

# ============ ЗАДАНИЕ 4 ============
man -k copy                  # или apropos copy
man -k archive               # найдёт tar, ar, etc.
apropos permission           # найдёт chmod, chown и др.

# ============ ЗАДАНИЕ 5 ============
cat > ~/cheatsheet.txt <<EOF
=== Шпаргалка по командам Linux ===

1. ps
   Источник: whatis
   Описание: $(whatis ps | head -1)
   Пример: ps aux

2. kill
   Источник: --help
   Синтаксис: kill [опции] <PID>
   Пример: kill -9 1234 (принудительное завершение)

3. tar
   Источник: man (раздел EXAMPLES)
   Пример архивации: tar -czf archive.tar.gz /path/to/dir
   (c - create, z - gzip, f - file)

4. sort
   Источник: info (раздел "Options")
   Основные опции:
   -n - числовая сортировка
   -r - обратный порядок
   -u - уникальные строки

Дата создания: $(date "+%Y-%m-%d %H:%M:%S")
EOF

cat ~/cheatsheet.txt        # проверка содержимого
```
