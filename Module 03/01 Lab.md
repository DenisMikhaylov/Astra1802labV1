
# Лабораторная работа №1: Основы работы в командной строке Linux

**Дисциплина:** Информационные технологии  
**Цель работы:** Получить базовые навыки работы в командной оболочке Linux (Bash): управление переменными, создание файлов, настройка алиасов, работа с историей команд и утилитой `date`.  
**Стенд:** Виртуальная машина Astra Linux.  
**Учетные данные:**  
- Пользователь: `sa`  
- Пароль: `12345678`  
- Имя машины: `Astra`

---

## Теоретическая справка

| Команда / Концепция | Описание |
|----------------------|----------|
| `echo $VAR` | Вывод значения переменной |
| `export VAR=value` | Создание глобальной переменной окружения |
| `touch file` | Создание пустого файла или обновление времени доступа |
| `cat > file <<EOF` | Создание файла с многострочным содержимым (здесь-документ) |
| `alias ll='ls -la'` | Создание псевдонима для команды |
| `history` | Просмотр истории выполненных команд |
| `!n` | Повторное выполнение команды под номером n из истории |
| `date` | Вывод текущей даты и времени |
| `date +%Y-%m-%d` | Вывод даты в заданном формате |

## Задания

### Задание 1. Работа с переменными
1. Войдите в систему как `sa` (пароль `12345678`).  
2. Создайте локальную переменную `STUDENT` и присвойте ей значение `"Иванов_Иван"`.  
3. Создайте глобальную переменную `GROUP` со значением `"ИТ-31"`.  
4. Выведите на экран значение обеих переменных одной командой.  
5. Создайте переменную `FULL_INFO`, которая объединяет значения `STUDENT` и `GROUP` через пробел.  
6. Запишите значение `FULL_INFO` в файл `~/info.txt`.

---

### Задание 2. Создание файлов на основе шаблона
1. Создайте в домашней директории каталог `lab_files`.  
2. Используя **здесь-документ** (heredoc), создайте файл `~/lab_files/report.txt` со следующим содержимым:
   ```
   Отчет по лабораторной работе
   Студент: Иванов Иван
   Группа: ИТ-31
   Дата выполнения: 14.07.2026
   ```
3. Создайте **пустой** файл `~/lab_files/empty.log`.  
4. Скопируйте `report.txt` в `~/lab_files/report_copy.txt`.

---

### Задание 3. Создание alias (псевдонимов)
1. Создайте временный алиас `ll` для команды `ls -la`.  
2. Создайте алиас `my_date` для команды `date +"%A, %d %B %Y"` (вывод дня недели, числа, месяца и года на русском).  
3. Проверьте работу обоих алиасов.  
4. Сделайте алиас `ll` **постоянным**, добавив его в файл `~/.bashrc`.

---

### Задание 4. Работа с историей команд
1. Выведите последние 5 команд из истории.  
2. Повторите **предпоследнюю** команду (без повторного ввода) с помощью истории.  
3. Найдите в истории команду, содержащую слово `report`, и выполните её (используйте `Ctrl+R` или `!<номер>`).  
4. Очистите историю сессии командой `history -c`.

---

### Задание 5. Работа с командой date
1. Выведите текущую дату в формате **ГГГГ-ММ-ДД ЧЧ:ММ:СС**.  
2. Выведите дату **через 3 дня** в формате `День_недели, ЧЧ:ММ`.  
3. Создайте файл с именем `backup_<текущая_дата>.tgz` (например, `backup_2026-07-14.tgz`) с помощью команды `touch` и подстановки результата `date`.

## Решение (эталон выполнения)

Ниже приведена последовательность команд, которые должен выполнить студент.

```bash
# ============ ЗАДАНИЕ 1 ============
# Вход в систему уже выполнен (sa / 12345678)

STUDENT="Иванов_Иван"
export GROUP="ИТ-31"
echo "Студент: $STUDENT, Группа: $GROUP"

FULL_INFO="$STUDENT $GROUP"
echo "$FULL_INFO" > ~/info.txt
cat ~/info.txt  # проверка

# ============ ЗАДАНИЕ 2 ============
mkdir ~/lab_files

cat > ~/lab_files/report.txt <<EOF
Отчет по лабораторной работе
Студент: Иванов Иван
Группа: ИТ-31
Дата выполнения: 14.07.2026
EOF

touch ~/lab_files/empty.log
cp ~/lab_files/report.txt ~/lab_files/report_copy.txt

# ============ ЗАДАНИЕ 3 ============
alias ll='ls -la'
alias my_date='date +"%A, %d %B %Y"'

ll                    # проверка
my_date               # проверка

echo "alias ll='ls -la'" >> ~/.bashrc   # делаем постоянным

# ============ ЗАДАНИЕ 4 ============
history 5             # последние 5 команд
!!                    # повторить последнюю (можно использовать !-2 для предпоследней)
!-2                   # выполнить предпоследнюю команду

# Поиск по истории (интерактивно Ctrl+R)
# Или если знаем номер: !<номер>

history -c            # очистка истории сессии

# ============ ЗАДАНИЕ 5 ============
date "+%Y-%m-%d %H:%M:%S"
date -d "+3 days" "+%A, %H:%M"
touch "backup_$(date +%Y-%m-%d).tgz"
# или через обратные кавычки: touch "backup_`date +%Y-%m-%d`.tgz"
```
